SkyprMC

This modpack is designed to enhance your Minecraft experience with new features and improvements. Explore new worlds, discover unique items, and enjoy a fresh take on the Minecraft gameplay you know and love.

PREREQUISITES:

Before installing the modpack, make sure you have the following installed:

  • Install 7zip - Needed to extract ZIP files.
  • Install Java 21 - Required to run Minecraft.
  • Install the latest Python OR from the Microsoft Store - The Fabric installer requires it.
  • Install Fabric Installer - Used to load the modpack into Minecraft.

These are required to ensure a successful installation of the modpack.
